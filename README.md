# 🍽️ Foodie Restaurant Website

A **responsive restaurant website** built using **HTML, CSS, and JavaScript**.  
This project shows a simple restaurant landing page with menu items and basic interactivity.

## 🚀 Features

- Responsive navigation bar
- Hero section
- Menu cards (Pizza, Burger, Pasta)
- Order button with JavaScript alert
- Restaurant open / close status
- Mobile responsive layout

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
